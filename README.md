# AindaDá

> Ainda dá pra usar?

SPA utilitária: item + data de abertura/validade → semáforo (ok / atenção / não dá) + dicas.

## Stack

- Next.js (App Router)
- Tailwind CSS v4
- Bun
- Seed JSON local (`src/data/seed.json`)

## Dev

```bash
cd ~/projects/aindada
bun dev
```
